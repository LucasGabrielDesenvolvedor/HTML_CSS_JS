Imagem dinâmica

Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador.

sempre que for usar a tag < source > sempre comece da maior para menor exemplo use a tag picture e dentro dela use a tag img escolhendo a maior imagem.

acima da tag img use a tag source escolhendo as dimensões da imagem média e logo em seguida use a tag source acima da primeira para colocar as dimensões da imagem mias pequena.

Reproduzindo áudio

vamos aprender a reproduzir áudios em html5

use as tags AUDIO e use os comandos CONTROLS e AUTOPLAY. dependedo do navegador o camndo autoplay ja faz com que o arquivo seja executado mas outros navegadores nao podem funcionar ai usamos o comando CONTROLS. Voce pode usar diferentes tipos de arquivos de audio mp3, wav, ogg.

outra maneira de reproduzir audio em html5

use a tag AUDIO e dentro dela use a tag SOURCE:SRC

caso este audio nao seja reproduzido o seu navegador não esta compativel a esta reprodução clique aqui para baixar o audio
